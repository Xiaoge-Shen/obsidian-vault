# ATNLP Mock Exams

This folder contains three original mock examination papers for INFR11287 Advanced Topics in Natural Language Processing.

They were generated from the 2026 exam guide, the selected NLU past-paper style examples, tutorials, coursework, labs, and local lecture revision materials.

## Files

- `mock-paper-1.pdf`: Retrieval, Question Answering, Data, Translation, and Evaluation
- `mock-paper-1-mark-scheme.pdf`: indicative answer outline for Paper 1
- `mock-paper-2.pdf`: Adaptation, Reasoning, Long Context, Summarisation, and Safety
- `mock-paper-2-mark-scheme.pdf`: indicative answer outline for Paper 2
- `mock-paper-3.pdf`: Multimodal Models, Agents, Interpretability, Parsing, and Ethics
- `mock-paper-3-mark-scheme.pdf`: indicative answer outline for Paper 3

The mark schemes include expanded explanations for every multiple-choice question, including the relevant concept being tested.

LaTeX source files are included beside each PDF so the papers can be edited and recompiled.

Additional guidance:

- `COVERAGE_AND_DIFFICULTY.md`: per-paper knowledge coverage, exam form, difficulty, and suggested practice order
- `VALIDATION.md`: build and acceptance checks

## Intended Use

Each paper is 50 marks, 3 compulsory questions, and designed for a 2-hour handwritten attempt. The style follows the relevant NLU past papers referenced in the ATNLP exam guide: scenario-based subquestions plus a focused multiple-choice section.

## Validation

See `VALIDATION.md` for the coverage and build checks.
